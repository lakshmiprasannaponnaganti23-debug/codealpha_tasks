# Banking System (C++)

## Description
This project is a simple C++ based Banking System that allows users to
create accounts, perform deposits, withdrawals, fund transfers, and
view transaction history using object-oriented programming concepts.

## Features
- Customer and Account management
- Deposit and withdrawal functionality
- Fund transfer between accounts
- Transaction history tracking
- Displays account balance and details
- Uses Object-Oriented Programming (OOP)

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling (optional)
- Standard Input / Output

## How to Run
1. Open any C++ compiler (Code::Blocks, Programiz, or OnlineGDB)
2. Copy the code from `banking.cpp`
3. Compile and run the program
4. Choose options from the menu to perform banking operations

## Files in This Repository
- `banking.cpp` – Main C++ source code
