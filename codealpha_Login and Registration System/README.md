# Login and Registration System (C++)

## Description
This project is a simple C++ based Login and Registration System.
It allows users to register with a username and password and then log in
using the same credentials. User data is stored in a file for persistence.

## Features
- User registration with username and password
- Duplicate username validation
- Secure storage of credentials using file handling
- Login authentication system
- Displays success and error messages
- Beginner-friendly C++ logic

## Technologies Used
- C++
- File Handling
- Standard Input / Output

## How to Run
1. Open any C++ compiler (Code::Blocks, Programiz, or OnlineGDB)
2. Copy the code from `login.cpp`
3. Compile and run the program
4. Choose Register or Login from the menu
5. Enter username and password as prompted
