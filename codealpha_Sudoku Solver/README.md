# Sudoku Solver (C++)

## Description
This project is a C++ program that solves a 9×9 Sudoku puzzle using the
backtracking algorithm. Empty cells are filled by trying valid numbers
according to Sudoku rules.

## Features
- Represents the Sudoku grid using a 2D array
- Uses backtracking and recursion
- Validates row, column, and 3×3 subgrid constraints
- Automatically solves incomplete Sudoku puzzles
- Displays the solved Sudoku grid

## Technologies Used
- C++
- Recursion
- Backtracking Algorithm
- Standard Input / Output

## How to Run
1. Open any C++ compiler (Code::Blocks, Programiz, or OnlineGDB)
2. Copy the code from `sudoku.cpp`
3. Compile and run the program
4. Enter the Sudoku puzzle (use 0 for empty cells)
5. View the solved Sudoku output
