# CGPA Calculator (C++)

## Description
This project is a simple C++ program that calculates the CGPA of a student
based on the number of courses, grade points, and credit hours.

## Features
- Takes user input for number of courses
- Accepts grade points and credit hours
- Calculates total credits
- Computes final CGPA
- Beginner-friendly C++ logic

## Technologies Used
- C++
- Standard Input / Output

## How to Run
1. Open any C++ compiler (Code::Blocks, Programiz, or OnlineGDB)
2. Copy the code from `cgpa.cpp`
3. Compile and run the program
4. Enter grade points and credit hours when prompted

## Files in This Repository
- `cgpa.cpp` – Main C++ source code
